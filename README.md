#Salsa Helpers
## Usage
```
/** @var \Drupal\salsa_helpers\SalsaHelpers $salsaHelpers */
$salsaHelpers = \Drupal::service('salsa.helpers');
```

### Some helper functions:
1. ```generateUuid()```
2. ```installModule()```
3. ```createEmptyBlockContent()```
4. ```createMenuLink()```
5. ```createTaxonomyTerm()```
